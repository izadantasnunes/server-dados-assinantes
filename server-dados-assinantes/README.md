# Server Dados Assinantes

Painel administrativo inspirado em softwares internos de banco de dados para controle de assinantes de jornal.

## Funcionalidades

- Menu lateral utilizável
- Abas: Estrutura, SQL, Pesquisar, Inserir, Exportar, Importar, Operações e Relatórios
- Ações funcionais: procurar, estrutura, pesquisar, inserir, limpar, eliminar, editar, copiar e excluir
- Cadastro de assinantes
- Edição de assinantes
- Exportação CSV
- Console SQL simulado
- Relatórios de receita, ticket médio e plano líder
- Dados salvos no LocalStorage do navegador

## Campos do assinante

- Nome
- Idade
- Endereço
- Telefone
- E-mail
- Data de ativação
- Período da assinatura
- Valor pago

## Tecnologias

- HTML
- CSS
- JavaScript
- SQL/MySQL

## Como abrir

Abra `index.html` no navegador ou use Live Server no VS Code.

O arquivo SQL está em:
